# Panel de Control de Análisis de Vehículos

Esta aplicación web permite analizar datos de anuncios de venta de coches 
de manera interactiva. Los usuarios pueden explorar diferentes aspectos 
del dataset a través de gráficos y filtros dinámicos; el usuario puede considerar precios y datos de odometro para su consulta.

En este proyecto se utilizaron las siguientes tecnologías: 
- Python
- Streamlit (aplicación web)
- Pandas (análisis de datos)
- Plotly (visualizaciones)
